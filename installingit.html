Github

On supported Windows or macOS operating systems, you can run GitHub Desktop.

For me, I recommend you guys to use Homebrew for the easiest and fastest way to intstall through 5 following steps below:

Navigate to Homebrew website.

Copy that command on that page, under Install Homebrew into the macOS Terminal.

Then enter your password and hit Return

Having completed, you will notice an installation successful message showing up.

Type that command in the Terminal and then you can install Git. Good job !

Installing GitHub

  • Go to the GitHub Desktop download page.
  • Click on Download for macOS 
  • In the Downloads folder of your machine, double-click the zip file of the GitHub Desktop.
  • After unzipping  the file, double-click  GitHub Desktop.
