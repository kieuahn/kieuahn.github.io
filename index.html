Step 0: Install git and create a GitHub account

The first two important things to do are install git and create a free GitHub account. Having done that, create a GitHub account here.



Step 1: Create usename and email

First, open the Terminal program on your computer and then: Check git config
Enter command : git config -l (to list everything)
Set up user.name + user.email
git config --global.username "yourname"
git config --global.email "@youremail"



Step 2: Create a Github repository

Click in the upper right-hand corner, next to your avatar or indenticon, and pick New Repository.
Name your repository.
Write a short description.
Select Initialize this repository with a README.
Click Create repository.

Step 3: Clone your repository to your local machine and make changes then upload it on your Github online

Having created a repositiory online, you open the terminal agian ang then type: git clone 'your Git URL'
git add .
git commit -m "write something meaningful here"
git push
