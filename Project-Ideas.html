Building an on-demand data lake in healthcare

This project is set out to implement, build and design a data search and analysis platform for healthcare organization to improve Healthcare Therapies and Plans.It is difficult for today's healthcare organizations to process and derive maximum value from the enormous data they gather, including clinical, genomics, medical and science data. The emergence of data lakes presents a cost-effective, scalable, and versatile way for the analysts and authorized user to get the best value from patient data. Applying companion tools and frameworks that improved researching and organizing process when setting up the data lake will allow the system to remain a crystal- clear pool of information for the organization for many years to come.

Motivation

The expanding advancement in technologies internationally comes with the growing demand of many business to sort, analyze and secure information and organizational data. Various businesses choose to have their data isolated in separate storage facilities: databases, data warehouses, and other storage systems around the enterprise. However, when it comes to having an enormous amount of dataset distributed in different formats, there are some practical difficulties in building up a data warehouse at a very initial stage for the company. One of the key reasons for this is that it is difficult to determine precisely which data sets are important and how they can be cleaned, enriched and converted from different systems to tackle difficult business problems. Without data science experts, pushing data into the warehouse could be time-consuming and easier to mess up. Investing in a data lake solves business and operational obstacles that have become difficult to tackle using conventional BIand data warehousing technology.

AWS EMR (Amazon Web Services and Elastic MapReduce)

EMR is Amazon's cloud platform that facilitates the collection of big data and data analytics, plus providing the businesses with a platform to host their data warehouses. A nice aspect of AWS EMR for healthcare is that it employs a standardized model for data management infrastructure and data analysis across multiple disconnected sources of health datasets.

Apache Spark

A cluster computing system that runs on Hadoop and handles various types of data. Spark has rich data handling capabilities and most significantly, is 10–20x faster than Hadoop's MapReduce. It also achieves these computational speeds through its in-memory primitives. The data is stored and display in the memory (RAM) and executes all in-memory computations. Spark is also an open-source analytics and data processing tool used for large-scale distributed datasets. Data scientists and developers use it to performETLjobs quickly on large-scale data fromIoTdevices, sensors and so on.
