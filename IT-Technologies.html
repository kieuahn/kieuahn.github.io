Cloud Computing

a.    Usage

According to Zaigham Mahmood [1], Cloud Computing is an enticing paradigm that enables customers to self-provide cloud-based tools, application services, platforms for development, and virtualized infrastructures, providing secure cloud and remote data center applications, hardware, and IaaS [2]. In order to execute complex large-scale computational operations, cloud platforms have become an influential architecture and cover a variety of IT features, from storage and computing to database and application services. Many companies and individuals have been motivated to implement cloud computing by the need to store, manage and interpret vast volumes of data sets [3]. 

 b.   The impact 

Cloud infrastructure is also a big opportunity for many third-world countries that have been left behind so far in the IT revolution, as some cloud storage vendors are leveraging the benefits of a cloud network to allow IT services in countries that have historically lacked the resources for the widespread implementation of IT services. When combine with other emerging fields it enables many new program classes possible and offers resources that were not accessible before. Some notable applications, namely, are GPU cloud computing platform [1] and mobile cloud computing [4] or even integrating Cloud Computing with E-Commerce technologies to create a big logistic system [5], which has been shown to significantly lessen the costs of the logistics sector up to 30%.Additionally, it significantly cuts the cost of entry for smaller companies looking to profit from computed-intensive market analytics that were previously available to the majority of the businesses. This computational task usually requires vast quantities of processing resources over a very limited period of time, and cloud computing makes such dynamic resource utilization feasible. Cloud computing can also reduce IT barriers to creativity, as can be seen by many promising start-ups, from ubiquitous online platforms such as Facebook and Youtube to more oriented applications such as TripIt (for travelling management) or Mint (for managing one's own finances). Cloud storage make it possible for businesses to scale up their offerings, constantly depending on reliable knowledge, on consumer demand. When computational resources are handled by applications, they can be implemented very easily as new requirements emerge. In reality, the aim of cloud computing is to scale resources up or down dynamically through software APIs, based on the client load, with minimal service provider involvement [6]
